# communication-protocols
This repository is about communication protocols for interfacing with devices using different boards and micro controllers.

## Micro controllers: 
- TM4C1294NCPDTI
- ATmega328P

## devices:
- ADXL345 accelerometer.
- LCD 16 x 2.

## Commuincation protocols:
- I2C
- SPI
- USART
- CAN
- LIN
